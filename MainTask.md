# 位图（JPG/PNG）转矢量图（SVG）Python 程序开发任务清单

## 项目目标

使用 Python 开发一个可运行的位图转矢量图程序，支持将 `jpg`、`jpeg`、`png` 输入图像转换为 `svg` 矢量图输出。

本项目优先完成以下能力：

1. 支持黑白/高对比图像的矢量化
2. 支持基础彩色图像的区域矢量化
3. 提供命令行接口
4. 输出结构清晰、可维护、可扩展的工程代码
5. 提供示例、测试、文档
